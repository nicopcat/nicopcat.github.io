# uniapp 是什么

> uni-app（uni，读 you ni，是统一的意思） 是一个使用 Vue.js 开发所有前端应用的框架，开发者编写一套代码，可发布到 iOS、Android、Web（响应式）、以及各种小程序、快应用等多个平台。

笑死，跟我一起念：**you ni ai pu!**

百度百科给的解释，可以说是相当百度了。

---

总结起来就这几点：

1. 这是一个 Vue 语法的前端框架
2. 多用于国内各种小程序和 H5
3. 坑非常多

说是一次开发，多端使用，但坑就坑在多端使用时，出了现各种不兼容的情况。
